INSERTION SORT - Ordenação por Inserção

Indicado para vetores que não sejam muito grandes. Começa a leitura dos dados sempres pela segunda posição do
vetor [1], faz a ordenação dos itens a esquerda. 
Um vetor com os seguintes valores: 30, 20, 10, 40 na primeira iteração o valor da variavel declarada será 20, 
pois está na segunda posição do vetor. Quando passamos para o segundo laço de repetição, 

	





Criando Vetor:

int nomeVetor[] = new int [4];
nomeVetor[0] = 3;
nomeVetor[1] = 5;
nomeVetor[2] = 8;
nomeVetor[3] = 2;


ou (forma mais rápida)
 
int nomeVetor[] = {1,2,3,4}; 


ou (apenas alterando o local do sinal do vetor '[]' - não muda em nada do modelo a cima)

int [] nomeVetor = {1,2,3,4};
